# Чек-лист: Общие указания → Позиции ВОР

## Правило
Каждое требование в общих указаниях = потенциальная позиция в ВОР.

## Авто-проверка

### Шаг 1: Найти все глаголы в общих указаниях
Искать: выполнить, устроить, защитить, произвести, осуществить, обеспечить, 
производить, заполнить, нарезать, окрасить, уплотнить, засыпать.

### Шаг 2: Преобразовать в позиции

| Глагол / Фраза | Позиция ВОР | Материалы | Код ГЭСН |
|----------------|------------|-----------|----------|
| "противоусадочные швы" | Нарезка швов в плите | — | УТОЧНИТЬ |
| "заполнить шва герметиком" | Заполнение шва герметиком | Полиуретановый герметик | УТОЧНИТЬ |
| "защитить закладные" | Окраска закладных с огрунтовкой | Грунт ЭП-057, Эмаль ХС 5132 | ГЭСН46-08-022-03 |
| "обратная засыпка ЩПС" | Засыпка вручную | ЩПС (фракция 0-20) | ГЭСН01-02-061 |
| "уплотнение слоями 300 мм" | Послойное уплотнение | — | ГЭСН01-02-061 |
| "Купл=0,95" | Уплотнение до коэф. 0,95 | — | Включено в ГЭСН |
| "ограждение котлована" | Временное ограждение | — | УТОЧНИТЬ |
| "отвод грунтовых вод" | Водопонижение / дренаж | — | УТОЧНИТЬ |
| "мониторинг геотехнический" | Геотехнический мониторинг | — | УТОЧНИТЬ |
| "бетонирование при t<0°С" | Утепление бетона / добавки | — | УТОЧНИТЬ |
| "консервация работ" | Защита конструкций | — | УТОЧНИТЬ |
| "деформационные швы 20мм" | Устройство деформационных швов | ППС15, герметик, трубы ПЭ | ГЭСН26-01-041-05 |
| "не допускается замачивание" | Контроль влажности (в ППР) | — | Не в ВОР |
| "послойное уплотнение" | Уплотнение грунта | — | Включено в засыпку |
| "утеплитель в основании" | Укладка утеплителя | Пенополистирол ППС25 | УТОЧНИТЬ |

### Шаг 3: Проверить наличие в ВОР

```markdown
- [ ] Противоусадочные швы → нарезка + герметик
- [ ] Деформационные швы → ППС + герметик + трубы ПЭ
- [ ] Защита закладных → грунт + эмаль + площадь окраски
- [ ] Обратная засыпка → ЩПС + уплотнение (Купл=0,95)
- [ ] Ограждение котлована → щиты / ограждение
- [ ] Дренаж / отвод воды → если указано в ППР
- [ ] Мониторинг → геодезия / контроль (в ППР)
- [ ] Утепление бетона → если t<0°С
- [ ] Консервация → защита конструкций (приостановка)
- [ ] Утеплитель в основании → ППС25 / ППС15
```

## Примеры из реальных проектов

### Проект КЖ6 (плиты на отм. 0,000)
**Общие указания листа 1:**
- "Противоусадочные швы выполнить путем нарезки плиты... на глубину 30 мм, шириной 5 мм"
  → Позиция: Нарезка швов 1728,8 м
  → Позиция: Герметик 300 кг
  
- "Все закладные изделия должны быть защищены в соответствии с листом общих данных"
  → Позиция: Окраска закладных (площадь по каждой марке)
  → Позиция: Грунт ЭП-057
  → Позиция: Эмаль ХС 5132

- "Материалы на заполнение деформационных швов учтены в спецификации"
  → Проверить: в спецификации есть трубы ПЭ? (да, 66+490 шт)

### Частые упущения
- [ ] Не учтены трубы ПЭ в деф.швах
- [ ] Не посчитана площадь окраски закладных
- [ ] Забыты противоусадочные швы (они НЕ на схемах!)
- [ ] Нет позиции "грунт" / "эмаль" — только "окраска"

## Вывод
**Если в общих указаниях есть ТРЕБОВАНИЕ — оно должно быть в ВОР.**
