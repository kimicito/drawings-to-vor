# Методология извлечения ВОР из чертежей

## 4-глазый метод (два прохода)

### Проход 1: Структурный (без цифр)

**Цель:** Понять, что на чертеже, не пытаясь извлечь количества.

**Что делать:**
1. Прочитать заголовок, штамп, масштаб
2. Найти легенду/условные обозначения
3. Перечислить типы элементов:
   - Стены, перекрытия, колонны (КР)
   - Двери, окна, перегородки (АР)
   - Вентиляция, отопление, водопровод (ОВ/ВК)
   - Электрика, слаботочка (ЭО/СС)
4. Найти спецификации оборудования (отдельные таблицы на листах)

**Выход:** Структурная карта — список элементов без чисел.

**Пример:**
```
ОВ-1, лист 3 (План 2 этажа, масштаб 1:100):
- Приточные камеры (2 шт — по плану видно)
- Воздуховоды прямоугольные (трассы по плану)
- Решетки жалюзийные (на отметках)
- Вентиляторы (в камерах, марки по спецификации)
- Спецификация оборудования (таблица внизу листа)
```

### Проход 2: Детальный (с цифрами)

**Цель:** Извлечь конкретные позиции с количествами.

**По каждому элементу из Прохода 1:**

#### Планы (АР, ОВ, ВК)
- Измерить на плане (в мм по линейке/экрану)
- Применить масштаб: `реальная длина = измеренная × масштаб`
- Пример: измерено 150 мм на плане, масштаб 1:100 → `150 × 100 = 15000 мм = 15 м`

#### Разрезы (КР, АР)
- Высоты, толщины — с разрезов
- Площади = длина × высота (с масштабом)
- Объёмы = площадь × толщина

#### Спецификации
- Копировать как есть (количества уже указаны)
- Сверить с планом: есть ли все позиции на чертеже?
- Сверить с планом: нет ли на чертеже позиций, которых нет в спецификации?

#### Схемы трубопроводов/воздуховодов
- Считать длины по линиям (с масштабом)
- Узлы = точки соединения, ответвления
- Оборудование = по спецификациям

### Таблица: откуда что берётся

| Тип чертежа | Что извлекаем | Откуда берём | Как считаем |
|-------------|--------------|--------------|-------------|
| План (АР) | Площади помещений | Габариты на плане | длина × ширина × масштаб² |
| План (АР) | Периметры | Габариты на плане | 2×(длина+ширина) × масштаб |
| Разрез (КР) | Объёмы бетона | Площадь сечения × длина | площадь × масштаб² × длина × масштаб |
| План (ОВ) | Длины воздуховодов | Линии на плане | длина на плане × масштаб |
| План (ОВ) | Кол-во решеток | Символы на плане | Считаем символы |
| Спецификация | Всё оборудование | Таблица на листе | Копируем как есть |
| Узел | Типы креплений | Детали узла | По типовым узлам |

## Критические ошибки (которые допускают все)

### 1. Забыть масштаб
**Последствие:** Все объёмы в 10×, 100× или 0.1× от реальных.  
**Решение:** Найти масштаб ПЕРЕД извлечением. Если нет — спросить.

### 2. Считать площадь вместо длины
**Последствие:** Воздуховоды в м² вместо пог.м.  
**Решение:** Проверить единицу измерения по ФЕР-2020.

### 3. Пропустить спецификацию
**Последствие:** Оборудование не попадёт в ВОР.  
**Решение:** Всегда проверять: есть ли спецификация на листе?

### 4. Дубли (позиция и на плане, и в спецификации)
**Последствие:** Двойной счёт в смете.  
**Решение:** Проверять: «Эта позиция уже есть?» Использовать validate_vor.py.

### 5. Не сверить спецификацию с планом
**Последствие:** Лишние или недостающие позиции.  
**Решение:** Фиксировать расхождения. Не «исправлять» самостоятельно.

## Качество данных

### Что делает извлечение сложным

| Фактор | Влияние | Решение |
|--------|---------|---------|
| Плохое качество скана | Не читаются размеры | Запросить лучшее качество |
| Нет масштаба | Невозможны объёмы | Запросить масштаб |
| Ручные пометки | Можно принять за оригинал | Игнорировать пометки, искать штамп |
| Несколько версий чертежей | Можно использовать устаревший | Проверить дату и ревизию |
| Сложные схемы | Много трасс, легко ошибиться | Разбивать на секции, считать по частям |

### Рекомендации по качеству сканов

- **DPI:** 200-300 для PDF, достаточно для чтения
- **Формат:** PDF с текстовым слоем (идеально) или PNG 300 DPI
- **Контраст:** Чёрный на белом, без серых полос
- **Полнота:** Все листы, включая спецификации и ведомости

## Примеры (из реальных проектов)

### Пример 1: Вентиляция (ОВ)

**Чертеж:** План 2 этажа, масштаб 1:100, спецификация оборудования внизу.

**Проход 1:**
- Приточная установка (1 шт — по спецификации)
- Воздуховоды прямоугольные (трассы по плану)
- Решетки жалюзийные (по плану — на стенах)
- Клапаны обратные (в спецификации)

**Проход 2:**
- Воздуховоды: измеряем трассы на плане, ×100 → пог.м
- Решетки: считаем символы на плане → шт
- Оборудование: копируем из спецификации → шт/компл

### Пример 2: Отопление (ВК)

**Чертеж:** Схема трубопроводов, масштаб 1:50.

**Проход 1:**
- Трубы стальные (трассы)
- Радиаторы (по плану)
- Краны, вентили (в спецификации)
- Узел теплового пункта (отдельный лист)

**Проход 2:**
- Трубы: измеряем трассы, ×50 → пог.м
- Радиаторы: считаем на плане → шт
- Арматура: из спецификации → шт
